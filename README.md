## ✨ 项目简介

欢迎来到 **CherryFC_F103** 的数字天空实验室！这是一个基于强大且经济高效的 **STM32F103** 微控制器所打造的下一代飞行控制器。我们致力于提供一个稳定、高性能且高度可定制的平台，让你的无人机梦想翱翔天际，无论是竞速无人机、航拍平台还是创新型飞行器，都能拥有卓越的表现。

Forget the ordinary. Embrace the extraordinary. 💫

## 🌟 核心特性

*   **极致性能**：利用 STM32F103 的强大处理能力，实现超快的姿态解算与响应速度。
*   **模块化设计**：清晰的代码结构，便于理解、修改和扩展，满足你的个性化需求。
*   **丰富接口**：支持多种传感器（陀螺仪、加速度计、磁力计等）、GPS、遥控器协议（SBUS, PPM, DSMX）以及更多外部模块。
*   **智能飞行模式**：预设多种飞行模式，从手动特技到稳定定高，满足不同飞行场景。
*   **实时调试**：便捷的串口输出与数据可视化支持，助你轻松调优。
*   **开源共享**：我们坚信开源的力量，欢迎社区的每一次贡献和创新！

## 🛠️ 快速开始

### 🚀 前置条件

在开始你的 CherryFC_F103 之旅前，请确保你已具备以下环境：

*   **STM32CubeIDE** 或其他兼容的 ARM GCC 开发环境
*   **ST-Link/J-Link** 调试器
*   **Python 3.x** (用于地面站或数据分析工具)

### ⚙️ 环境配置

1.  **克隆仓库**：

    ```bash
    git clone https://github.com/your_username/CherryFC_F103.git
    cd CherryFC_F103
    ```

2.  **打开项目**：
    使用 STM32CubeIDE 导入项目文件。

3.  **编译与烧录**：
    -   根据你的硬件配置调整 `config.h` (如果存在)。
    -   编译项目。
    -   通过调试器将固件烧录到你的 STM32F103 开发板。

### 📡 连接地面站 (可选)

我们正在开发一个功能强大的地面站，用于实时监控、参数配置和固件升级。敬请期待！✨
